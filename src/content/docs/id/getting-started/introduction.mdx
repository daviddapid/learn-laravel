## ❓ Apa itu Laravel & Mengapa Laravel?

Laravel adalah salah satu framework PHP yang sangat populer untuk membangun aplikasi web. Syntax nya yang cantik, tools nya yang kuat, dan ekosistem nya yang lengkap membantu developer untuk membuat aplikasi web.

<Image
  src={statisticImg}
  alt={"Statistic most popular backend frameworks"}
  loading={"eager"}
/>
Perlu diketahui bahwa Laravel bukan hanya framework untuk satu tugas (frontend/backend) saja, tetapi ia adalah Fullstack Framework. Artinya kita dapat membuat suatu aplikasi web yang utuh hanya dengan menggunakan Laravel. Cek [Laravel di GitHub](https://github.com/laravel/laravel) — **80k**🌟 stars.

<Aside type='note' title='Frontend, Backend ??'> 
Jadi dalam pengembangan aplikasi web, terdapat 2 komponen utama yaitu: 
1. **Frontend**, ini berkaitan dengan tampilan/ui/apapun yang bisa dilihat oleh user dengan kasat mata dari aplikasi web yang kita buat. seperti: button, page, avatar
2. **Backend**, ini berkaitan dengan bagaimana aplikasi web bekerja (sistem di balik layar). seperti: Proses order produk, pembayaran, CRUD produk, dsb.

Simpelnya Frontend adalah tampilan dari aplikasi mu, sedangkan Backend adalah sistem dari aplikasi mu. Sedangkan **Fullstack** adalah gabungan dari keduanya
</Aside>
